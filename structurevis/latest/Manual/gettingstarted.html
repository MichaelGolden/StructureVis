StructureVis User Documentation


Introduction

StructureVis is a tool for visualising RNA and DNA secondary structures.

Platforms supported

StructureVis has been developed for use on Windows, but should run on Linux and Mac (not yet tested).

Requirements

Jave Runtime Environment (JRE) version 1.6.0 or greater is required to run StructureVis and at least 1GB of memory is recommended.

Installing StructureVis

StructureVis is designed to be portable and therefore does not come with an installer, in order to use it in on your computer, simply extract the StructureVis.zip to the location of your choice. This is a cross-platform installation.

Running StructureVis

In order to run StructureVis navigate to you StructureVis directory and launch the 'StructureVis.jar' file. If you are using Windows you can double-click the 'StructureVis_windows.exe' executable.
